  1. array在使用的时候需要初始化，不然会出错, 部分值会被初始化成奇怪的值。 原因:

    定义于任何函数体之外的变量被初始化为0。定义在函数体内部的内置变量将不被初始化（uninitialized）。 一个未被初始化的内置类型变量的值是未定义的。此时， 程序可能继续工作、可能崩溃，也可能生成垃圾数据。( P41 & P33, C++ Primer )

  1. 函数void * memset ( void * ptr, int value, size_t num );在使用的时候要注意第三个参数的值。这三个参数的定义在下面的表格里。
Params Definition
ptr Pointer to the block of memory to fill.
value Value to be set. The value is passed as an int, but the function fills the block of memory using the unsigned char conversion of this value.
num Number of bytes to be set to the value. size_t is an unsigned integral type.

  4. 一些实用的函数总结。

ctype.h 里的一些函数。

  • isalnum: Checks whether c is either a decimal digit or an uppercase or lowercase letter. isalpha or isdigit is true.
  • isalpha: Checks whether c is an alphabetic letter.
  • isdigit: Check if character is decimal digit.
  • tolower: Converts c to its lowercase equivalent if c is an uppercase letter and has a lowercase equivalent. If no such conversion is possible, the value returned is c unchanged.
  • toupper: Converts c to its uppercase equivalent if c is a lowercase letter and has an uppercase equivalent. If no such conversion is possible, the value returned is c unchanged.
    string里的一些函数。

  1. switch只能用数字或者枚举。

  2. 关于设置小数点之后的位数的精度。有两种方法。

    • 使用library iomanip, 在使用的过程中可以用fixed和setprecision来定小数位数。如果不加fixed的话，就说明总位数为多少位，加了fixed则是小数位数多少位。
    • 使用librarystdio.h, 用printf来输出,相应的，读取就要使用scanf。 另外，读取double格式的数用的是 scanf("%lf", &r);

  3. getline。可以用来读取一行string，getline(cin, str);

  4. end of string character 有的时候需要注意，因为这预示着string的结束。

  5. 主要边际条件，不要漏掉。看这题参考一下。ITP1_9_A

  6. 使用string的时候，可以使用函数 string substr (size_t pos = 0, size_t len = npos) const;， 末尾不写默认到最后。

  7. getline和cin混用的时候需要注意。流提取运算符根据它后面的变量类型读取数据，从非空白符号开始，遇到Enter、Space、Tab键时结束。getline函数从istream中读取一行数据，当遇到“\n”时结束返回。因此在使用getline函数之前要注意，输入流中是否有可能会有换行符，如果有则使用cin.ignore清除。

  8. c++中的sin和cos要用pi来算。

  9. int在使用的时候也要记得初始化。

  10. Overflow: answer will in the range of 32-bit signed integer.

  • 32-bit signed integer: -2^31~2^31
  • 32-bit integer: 0~2^32
